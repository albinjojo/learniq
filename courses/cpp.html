C++ Basics for Beginners

C++ is a powerful general-purpose programming language that supports both procedural and object-oriented programming. This guide will help you understand the fundamentals of C++ programming.

Introduction to C++

C++ is an extension of the C programming language that adds support for object-oriented programming. It's widely used in system software, game development, and high-performance applications.

#include using namespace std; int main() { cout << "Hello, World!" << endl; return 0; }

Variables and Data Types

C++ has several built-in data types for storing different kinds of data:

Data Type Description Example
int Integer numbers int age = 25;
float Single precision floating point float price = 19.99f;
double Double precision floating point double pi = 3.14159;
char Single character char grade = 'A';
bool Boolean values bool isStudent = true;

Functions

Functions are reusable blocks of code that perform specific tasks:

// Function declaration int add(int a, int b) { return a + b; } // Function with default parameters void greet(string name = "User") { cout << "Hello, " << name << "!" << endl; }

Control Flow

C++ provides various control flow statements:

// if-else statement if (age >= 18) { cout << "You are an adult" << endl; } else { cout << "You are a minor" << endl; } // for loop for (int i = 0; i < 5; i++) { cout << i << endl; } // while loop int count = 0; while (count < 3) { cout << count << endl; count++; }

Arrays and Strings

C++ provides built-in support for arrays and string manipulation:

// Array declaration and initialization int numbers[5] = {1, 2, 3, 4, 5}; // String manipulation string name = "John"; name += " Doe"; // String concatenation cout << name.length() << endl; // String length

Classes and Objects

C++ supports object-oriented programming through classes:

class Person { private: string name; int age; public: Person(string n, int a) { name = n; age = a; } void display() { cout << "Name: " << name << ", Age: " << age << endl; } };

Pointers and References

C++ provides powerful memory management features:

// Pointer declaration and usage int x = 10; int* ptr = &x; cout << *ptr << endl; // Dereferencing // Reference declaration int& ref = x; cout << ref << endl; // Using reference

Exception Handling

C++ provides try-catch blocks for error handling:

try { // Code that might throw an exception throw runtime_error("An error occurred"); } catch (const exception& e) { cout << "Error: " << e.what() << endl; }

Conclusion

C++ is a powerful and versatile programming language that forms the foundation of many software systems. Understanding these basics will help you build efficient and robust applications.

FAQs

Q: What is C++?
A: C++ is a general-purpose programming language that extends C with object-oriented features.

Q: What are the main features of C++?
A: C++ features include object-oriented programming, templates, exception handling, and low-level memory manipulation.

Q: What is the difference between C and C++?
A: C++ adds object-oriented programming features, templates, and exception handling to C.
