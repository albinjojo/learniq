Python Basics for Beginners

Python is a versatile, high-level programming language known for its simplicity and readability. This guide will help you understand the fundamentals of Python programming.

Introduction to Python

Python is an interpreted, high-level programming language that emphasizes code readability with its notable use of significant whitespace. It's widely used in web development, data science, artificial intelligence, and more.

# Your first Python program print("Hello, World!")

Variables and Data Types

Python has several built-in data types for storing different kinds of data:

Data Type Description Example
int Integer numbers age = 25
float Floating point numbers price = 19.99
str Text strings name = "John"
bool Boolean values is_student = True
list Mutable sequence of items fruits = ["apple", "banana"]
dict Key-value pairs person = {"name": "John", "age": 25}

Functions

Functions are reusable blocks of code that perform specific tasks:

# Function definition def greet(name): return f"Hello, {name}!" # Function with default parameters def greet_user(name="User"): print(f"Hello, {name}!") # Lambda function square = lambda x: x ** 2

Control Flow

Python provides various control flow statements:

# if-else statement if age >= 18: print("You are an adult") else: print("You are a minor") # for loop for i in range(5): print(i) # while loop count = 0 while count < 3: print(count) count += 1

Lists and Dictionaries

Python provides powerful data structures:

# List operations fruits = ["apple", "banana", "orange"] fruits.append("grape") # Add item fruits.remove("banana") # Remove item print(fruits[0]) # Access item # Dictionary operations person = { "name": "John", "age": 25, "city": "New York" } print(person["name"]) # Access value person["email"] = "john@example.com" # Add key-value pair

Classes and Objects

Python supports object-oriented programming:

class Person: def __init__(self, name, age): self.name = name self.age = age def display(self): print(f"Name: {self.name}, Age: {self.age}") # Create object person = Person("John", 25) person.display()

File Handling

Python makes it easy to work with files:

# Reading from file with open("file.txt", "r") as file: content = file.read() print(content) # Writing to file with open("file.txt", "w") as file: file.write("Hello, World!")

Exception Handling

Python provides try-except blocks for error handling:

try: # Code that might raise an exception result = 10 / 0 except ZeroDivisionError: print("Cannot divide by zero") except Exception as e: print(f"An error occurred: {e}")

Modules and Packages

Python has a rich ecosystem of modules and packages:

# Importing modules import math import random # Using module functions print(math.pi) print(random.randint(1, 10))

Conclusion

Python is a versatile and beginner-friendly programming language that's widely used in various fields. Understanding these basics will help you build a strong foundation for your programming journey.

FAQs

Q: What is Python?
A: Python is a high-level, interpreted programming language known for its simplicity and readability.

Q: What are Python's main features?
A: Python features include dynamic typing, automatic memory management, and a comprehensive standard library.

Q: What can you build with Python?
A: Python can be used for web development, data science, artificial intelligence, automation, and more.
