CSS Styles Guide – A Complete Reference

Last Updated : 03 Dec, 2024

CSS (Cascading Style Sheets) is the language used to style and layout web pages. It describes how HTML elements should be displayed and allows you to create responsive and visually appealing websites.

Introduction to CSS

CSS works by selecting HTML elements and applying styles to them. There are three ways to add CSS to your HTML:

1. Inline CSS: Using the style attribute

2. Internal CSS: Using the <style> tag in the head section

3. External CSS: Using a separate .css file

CSS Selectors

Selector Description Example
element Selects all elements of the specified type p { color: blue; }
.class Selects elements with the specified class .highlight { background: yellow; }
#id Selects element with the specified ID #header { font-size: 24px; }
* Universal selector - selects all elements * { margin: 0; }
element element Descendant selector - matches all instances of element inside element div p { margin: 10px; }
element > element Child selector - selects direct children only div > p { margin: 10px; }
element + element Adjacent sibling selector h1 + p { margin-top: 20px; }
:hover Selects element when mouse hovers over it button:hover { background: blue; }
::before Creates a pseudo-element before the content p::before { content: "→"; }
::after Creates a pseudo-element after the content p::after { content: "←"; }

Common CSS Properties

Property Description Example
color Sets the text color color: #00A676;
background Sets all background properties background: #f5f5f5 url('bg.jpg') no-repeat center;
font-family Sets the font family for text font-family: 'Arial', sans-serif;
font-size Sets the font size font-size: 16px;
font-weight Sets the thickness of text font-weight: bold;
text-align Sets text alignment text-align: center;
margin Sets outer spacing margin: 10px 20px;
padding Sets inner spacing padding: 15px;
border Sets element borders border: 1px solid #ddd;
border-radius Sets rounded corners border-radius: 8px;
display Sets how an element should be displayed display: flex;
position Sets positioning method position: relative;
top, right, bottom, left Sets position coordinates top: 0; left: 0;
z-index Sets stack order of elements z-index: 100;
opacity Sets element transparency opacity: 0.8;
transform Applies 2D or 3D transformations transform: rotate(45deg);
transition Sets transition effects transition: all 0.3s ease;
animation Sets animation properties animation: slide 1s ease infinite;

The Box Model

The CSS box model is the foundation of layout on the web. Every element is represented as a rectangular box, with content, padding, border, and margin.

.box { content: "actual content"; padding: 20px; border: 1px solid black; margin: 10px; }

Layout Properties

Property Description Example
flex Creates flexible layouts display: flex;
grid Creates grid layouts display: grid;
float Floats elements left or right float: left;

Responsive Design

CSS provides several tools for creating responsive designs that adapt to different screen sizes:

@media (max-width: 768px) { .container { width: 100%; padding: 10px; } }

Conclusion

CSS is a powerful language that gives you complete control over the look and feel of your website. Understanding these fundamental concepts will help you create beautiful and responsive web designs.

CSS – FAQs

What is CSS specificity?

CSS specificity is the set of rules used to determine which style declarations are ultimately applied to an element. The more specific a selector is, the higher its precedence.

What are CSS preprocessors?

CSS preprocessors like Sass and Less extend CSS with additional features like variables, nested rules, mixins, and functions, making stylesheets more maintainable.

What is the difference between margin and padding?

Margin creates space outside an element's border, while padding creates space inside the border, between the border and the content.
