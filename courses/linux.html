Basic Linux Commands For Beginners [2025]

Last Updated : 19 March, 2025

While performing a task, we all need shortcuts. Shortcuts help us to complete a task quickly. Linux comes with such commands which are one to two words, using that commands, you can perform several operations in no time. As a beginner, you must be aware of those basic Linux commands to complete an operation in a short time in the Linux-based operating system.

What is Linux?

Linux is a free and open-source software that operates on its own operating system. The term 'Linux' stands for GNU + Linux. Initially developed by Linus Torvalds, it was created alongside the source code of Unix. While Linux is extensively utilized for various purposes, its applications are well-known to many.

Use of Linux Commands

Linux commands are a type of Unix command or shell procedure. They are the basic tools used to interact with Linux on an individual level. Linux commands are used to perform a variety of tasks, including displaying information about files and directories.

Linux operating system is used on servers, desktops, and maybe even your smartphone. It has a lot of command line tools that can be used for virtually everything on the system.

Most-Commonly Used Linux Commands

1. ls command in Linux

The ls command is commonly used to identify the files and directories in the working directory. This command is one of the many often-used Linux commands that you should know.

ls

This command can be used by itself without any arguments and it will provide us the output with all the details about the files and the directories in the current working directory. There is a lot of flexibility offered by this command in terms of displaying data in the output.

2. pwd command in Linux

The pwd command is mostly used to print the current working directory on your terminal. It is also one of the most commonly used commands.

pwd

Now, your terminal prompt should usually include the entire directory. If it doesn't, this is a quick command to see which directory you're in. Another purpose for this command is when creating scripts because it can help us find the directory in which the script was saved.

3. mkdir command in Linux

This mkdir command allows you to create fresh directories in the terminal itself. The default syntax is mkdir <directory name> and the new directory will be created.

mkdir GeeksforGeeks

4. cd command in Linux

The cd command is used to navigate between directories. It requires either the full path or the directory name, depending on your current working directory. If you run this command without any options, it will take you to your home folder. Keep in mind that it can only be executed by users with sudo privileges.

cd directory_name

5. rmdir command in Linux

The rmdir command is used to delete permanently an empty directory. To perform this command the user running this command must be having sudo privileges in the parent directory.

rmdir directory_name

6. cp command in Linux

The cp command is used to copy files or directories. It creates a copy of the source file or directory at the specified destination. This command requires read permissions for the source and write permissions for the destination.

cp source_file destination_file

7. mv command in Linux

The mv command is used to move or rename files and directories. It can move a single file or multiple files to a destination directory, or rename a file or directory.

mv source_file destination_file

8. rm command in Linux

The rm command is used to remove files or directories. It can delete multiple files at once and has options for recursive deletion of directories. Be careful with this command as it permanently deletes files.

rm file_name

9. uname command in Linux

The uname command displays system information. It can show the operating system name, system hostname, kernel release date, version, hardware platform, and processor information.

uname [options]

10. locate command in Linux

The locate command is used to find files by name. It searches through a pre-built database of files and directories, making it faster than the find command for simple searches. The database is typically updated daily.

locate file_name

11. touch command in Linux

The touch command is used to create empty files or update the timestamp of existing files. It's particularly useful when you need to create multiple files at once or update file access times.

touch file_name

12. ln command in Linux

The ln command is used to create links between files. It can create both hard links and symbolic links (symlinks). Hard links are exact copies of the original file, while symlinks are pointers to the original file.

ln -s source_file link_name

13. cat command in Linux

The cat command is used to view, create, and concatenate files. It can display file contents, combine multiple files, and create new files. It's one of the most frequently used commands for file operations.

cat file_name

14. clear command in Linux

The clear command is used to clear the terminal screen. It removes all previous commands and output, giving you a clean workspace. This command is equivalent to pressing Ctrl+L in most terminals.

clear

15. ps command in Linux

The ps command displays information about running processes. It shows process IDs, CPU usage, memory usage, and other details about active processes. It's essential for system monitoring and process management.

ps [options]
